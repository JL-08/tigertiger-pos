## Description
Point-of-Sale System for Tiger Tiger Boba MilkTea Shop.

## Installation (Development)

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Endpoints

1. Install the insomnia app
2. Import the `endpoints.json` file
3. Enjoy
